
# language-culture-names

  All language culture names.

## Example

```js
var names = require('language-culture-names');
console.log(names);
// => [
//      'af-ZA',
//      'sq-AL',
//      'ar-DZ',
//      ...
```

## Installation

```bash
$ npm install language-culture-names
```
